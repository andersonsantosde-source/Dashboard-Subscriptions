# Dashboard de Subscrições Xbox

Este projeto é um dashboard interativo para análise de dados de subscrições da Xbox, desenvolvido como parte do desafio da Digital Innovation One (DIO).

## Descrição

O dashboard permite visualizar e analisar métricas importantes sobre os subscritores da Xbox, incluindo planos de subscrição, passes sazonais (EA Play e Minecraft), valores de cupons e receitas totais. Utiliza uma paleta de cores verde inspirada na identidade visual da Xbox.

## Dados

Os dados utilizados estão no arquivo `Xbox_subscriptions.xlsx`, contendo as seguintes informações:

- **Subscriber ID**: Identificador único do subscritor
- **Name**: Nome do subscritor
- **Plan**: Plano de subscrição
- **Start Date**: Data de início da subscrição
- **Auto Renewal**: Renovação automática (Sim/Não)
- **Subscription Price**: Preço da subscrição
- **Subscription Type**: Tipo de subscrição
- **EA Play Season Pass**: Se possui passe EA Play
- **EA Play Season Pass Price**: Preço do passe EA Play
- **Minecraft Season Pass**: Se possui passe Minecraft
- **Minecraft Season Pass Price**: Preço do passe Minecraft
- **Coupon Value**: Valor do cupom aplicado
- **Total Value**: Valor total pago

O dataset contém 295 registros de subscritores.

## Funcionalidades do Dashboard

- Visualização de métricas gerais (número de subscritores, receita total)
- Análise por tipo de plano e subscrição
- Distribuição de passes sazonais
- Impacto de cupons na receita
- Tendências temporais de subscrições

## Tecnologias Utilizadas

- Excel/Power BI (para criação do dashboard)
- Dados estruturados em planilhas

## Como Usar

1. Abra o arquivo `Xbox_subscriptions.xlsx`
2. Navegue para a aba "Dashboard" para visualizar as análises
3. Utilize os filtros disponíveis para explorar os dados
